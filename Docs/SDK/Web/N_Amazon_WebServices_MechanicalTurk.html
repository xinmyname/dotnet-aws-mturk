SDK core classes that enable messaging between the client application and the Mechanical Turk web service


Classes

  Class Description
MTurkClient Core client used to invoke soap operations on the Mechanical Turk endpoint with a set of client credentials.
MTurkConfig Holds the client configuration necessary to communicate with the mechanical turk webservice. A MTurkConfig instance must be passed to the SimpleClient or MTurkClient instance to establish connectivity.

The DefaultInstance can be used when the configuration is retrieved from the application configuration file. If the configuration is stored elsewhere, a separate MTurkConfig instance must be instantiated and passed to the requester.
