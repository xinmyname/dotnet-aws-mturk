Logger used by the SDK to log events. This is a static class (Module in Visual Basic)static classModule and so cannot be inherited or instantiated.

Public Properties

  Name Description
Level Defined log levels are:
  • 1 - Debug (messages for debugging an application)
  • 2 - Info (messages that might be of interest to an application administrator)
  • 3 - Warn (messages that might indicate a potential problem)
  • 4 - Error (messages that indicate an error)
  • 5 - None (no logging)
Logger Sets the logger used to log SDK messages. By default, the a trace logger is used, but any other class implementing the ILog can be set here to replace this default logger. As such it is easy to hook in other log mechanisms and frameworks such as log4net, the Microsoft logging application block or simply the event log.
