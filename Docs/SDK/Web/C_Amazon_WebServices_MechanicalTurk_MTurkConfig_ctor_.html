Remarks

The following values are read from the application configuration file:

  • MechanicalTurk.ServiceEndpoint - The service endpoint receiving and processing mturk requests. Defaults to the sandbox endpoint at https://mechanicalturk.sandbox.amazonaws.com?Service=AWSMechanicalTurkRequester
  • MechanicalTurk.AccessKeyId - Required: The access key ID for the Amazon Web Services account
  • MechanicalTurk.SecretAccessKey - Required: The secret access key for the Amazon Web Services account
  • MechanicalTurk.Log.Level - The log level for SDK log messages (ILog for details). Defaults to 2 (Info)
  • MechanicalTurk.Proxy.Url - Proxy URL (only necessary when used through a web proxy that requires authentication)
  • MechanicalTurk.Proxy.User - User name for proxy credentials (only necessary when used through a web proxy that requires authentication)
  • MechanicalTurk.Proxy.Password - Password for proxy credentials (only necessary when used through a web proxy that requires authentication)
  • MechanicalTurk.Proxy.Domain - User domain for proxy credentials (only necessary when used through a web proxy that requires authentication)
  • MechanicalTurk.MiscOptions.EnsureQuestionsAreXmlWrapped - If set to true (default), ensures that questions are wrapped as XML when sent to Mechanical Turk.
  • MechanicalTurk.MiscOptions.EnsureQuestionValidity - If set to true (default is false), ensures that questions are validated before they are sent to Mechanical Turk
