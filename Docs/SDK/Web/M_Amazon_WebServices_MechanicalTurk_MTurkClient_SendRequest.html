Remarks

This method introduces flexibility in how requests can be sent to the Mechanical Turk Web Service.

Example

The following code samples show the creation of a HIT in 4 different ways (single or batch creation)

Method 1: Use of the convenience method

 Copy Code 
            CreateHITRequest req = new CreateHITRequest();            
            req.Title = string.Format("{0} (Test suite: {1})", title, DateTime.Now.ToString("g"));
            ...
            CreateHIT hit = new CreateHIT();
            hit.Request = new CreateHITRequest[] { req };
            CreateHITResponse response = TestUtil.Client.CreateHIT(hit);

Method 2: Use of the generic send method to create a single HIT

 Copy Code 
            CreateHITRequest req = new CreateHITRequest();            
            req.Title = string.Format("{0} (Test suite: {1})", title, DateTime.Now.ToString("g"));
            ...
            CreateHITResponse response = (CreateHITResponse)TestUtil.this.SendRequest(req);

Method 3: Use of the generic send method to create a multiple HITs

 Copy Code 
            CreateHITRequest req1 = new CreateHITRequest();            
            req1.Title = string.Format("{0} (Test suite: {1})", title, DateTime.Now.ToString("g"));
            ...
            CreateHITRequest[] reqArray = new CreateHITRequest[] { req1, req2, ..., reqX };
            CreateHITResponse response = (CreateHITResponse)TestUtil.this.SendRequest(reqArray);

Method 4: Use of the generic send method to create a multiple HITs utilizing generics

 Copy Code 
            List<CreateHITRequest> items = new List<CreateHITRequest>;
            CreateHITRequest req = new CreateHITRequest();            
            req.Title = string.Format("{0} (Test suite: {1})", title, DateTime.Now.ToString("g"));
            ...
            items.Add(req);
            ...
            CreateHITResponse response = (CreateHITResponse)TestUtil.this.SendRequest(items.ToArray());
