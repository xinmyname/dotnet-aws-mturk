Read Me for the Amazon Mechanical Turk SDK for .Net v1.5.0

The Amazon Mechanical Turk .Net SDK is a set of libraries and tools designed to make it easier for you to build solutions leveraging Amazon Mechanical Turk. The goals of the SDK are:

  • To abstract you from the "muck" of using web services
  • To simplify using the various Amazon Mechanical Turk APIs
  • To allow you to focus more on solving the business problem and less on managing technical details

Version Information

This distributable contains the Amazon Mechanical Turk SDK for .Net version 1.5.0.

The SDK is compatible with Amazon Mechanical Turk web service version 2011-10-01.

SDK Contents

The SDK distributable contains the following content:
  • Overview of the Amazon Mechanical Turk SDK for .Net to help you get started with the SDK.
  • Pre-compiled binaries ready to be used for development.
  • Set of sample applications demonstrating various aspects of the SDK and Amazon Mechanical Turk.
  • MSDN style documentation for SDK
  • Source code for SDK and Unit tests
